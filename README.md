# fortran-cli-ninja-hello-world

## Description
This is a POC project for fortran.
Uses ninja file with  gfortran compiler.

## Tech stack
- fortran
- gfortran
- g++
- ninja

## Docker stack
- alpine:edge

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
